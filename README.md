# Cloudflare AdBlock Worker

یک ورکر ساده، سبک و کاربردی برای بلاک‌کردن تبلیغات مزاحم روی موبایل — مخصوصاً تبلیغات «تپسل» و «یکتانت» که معمولاً در مرورگر موبایل (Chrome/Android) اذیت می‌کنند.

این پروژه کمک میکنه تا **حدود ۸۰٪ تبلیغات تپسل، یکتانت و سرویس‌های مشابه** بلاک بشن. فقط کافیه ورکر را روی Cloudflare فعال کنی و توسط اپ Intra یا هر DNS-over-HTTPS کلاینت دیگری تنظیم کنی.

---

## ✨ این ورکر چکار می‌کنه؟

* هر درخواستی که توی URL آن کلمه‌های `tapsell`، `yektanet`، `mediaad` یا… وجود داشته باشه **مسدود می‌شود**.
* سرعت خوبی داره.
* مناسب کسانی که حالشون از تبلیغات بی سرته بهم میخوره.

---

## 🚀 طرز استفاده (خیلی ساده)

1. برو به Cloudflare → Workers
2. یک ورکر جدید بساز
3. محتوای فایل `worker.js` همین ریپو را کامل کپی کن
4. ورکر را Deploy کن
5. آدرس ورکر را در برنامه Intra وارد کن (به عنوان DNS-over-HTTPS)
6. تمام! تبلیغات مزاحم تا حد زیادی حذف میشن 😄

---

## 🔧 چطور کار می‌کند؟

ورکر هر درخواست ورودی را چک می‌کنه و اگر لینک شامل یکی از کلمات مسدودی باشد، پاسخ `403 Blocked` می‌دهد. اگر نبود، درخواست را عادی عبور می‌دهد.

این کار باعث می‌شه فایل‌ها، ویدیوها، اسکریپت‌ها و منابع مربوط به تبلیغات اصلاً از سرور بارگذاری نشوند.
